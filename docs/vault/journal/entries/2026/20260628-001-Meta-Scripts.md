# 20260628-001 — Meta Scripts Archive Loop

## Goal

Stop agents from repeatedly archiving the Meta Scripts project.

## Problem

The file had `status: abandoned` and "Abandoned" in the title since vault import (`382849c`). Every agent read that, trusted it, and moved it to `archive/`. Happened twice across two different days.

## Resolution

- Restored file from `archive/` → `projects/`
- Rewrote frontmatter: removed `archive` tag, changed to `projects` tag, stripped abandoned language
- Rewrote body content to reflect active in-progress status
- Fixed 6 cross-references across map, projects README, scripts README, knowledge-audit, CHANGELOG
- Added explicit DO NOT ARCHIVE rule to AGENTS.md conventions

## Lesson

Frontmatter is executable knowledge. Agents trust metadata absolutely. Stale metadata fields cause automated actions that are hard to undo.
