# 20260605-001 -- CUDA 12.4 Dual-Arch Build

## Goal

Build llama.cpp with dual-GPU support (RTX 3060 sm_86 plus Tesla P40 sm_61).

## Problem

Default llama.cpp build targets one architecture. P40 kernel module would not load via modprobe.d.

## Resolution

Built with CMAKE_CUDA_ARCHITECTURES=61;86. Fixed P40 with kernel cmdline parameter instead of modprobe.d. Power cable troubleshooting added to config-notes.md.

## Decisions

Kernel cmdline wins over modprobe.d for driver parameters -- survives module rebuilds.
