# 🧪 Experimental Hats

Unstable, unconventional, or uncensored roles.  
These may produce unexpected output — use intentionally, not as defaults.

## Usage

Same as core hats: open the file, copy the prompt block, paste into system prompt.  
Some experimental hats intentionally bypass safety layers — be aware of context.

## The Hats

| # | Hat | When to use |
|---|-----|-------------|
| E1 | [[experimental/hat-creative]] | Fiction, worldbuilding, character dialogue, poetry |
| E2 | [[experimental/hat-social]] | Mediation, coaching, conflict-resolution, emotional intelligence |
| E3 | [[experimental/hat-adversarial]] | Devil's advocate, red-team, stress-test ideas |
| E4 | [[experimental/hat-unfiltered]] | Uncensored — no rules, no filters, full freedom |
| E5 | [[experimental/hat-persona]] | Deep roleplay — stay in character across entire session |
| E6 | [[experimental/hat-brainstorm]] | Divergent ideation — quantity over quality, zero judgment |
| E7 | [[experimental/hat-tactical]] | Wargaming, opponent simulation, competitive optimization |
| E8 | [[experimental/hat-socratic]] | Answers with questions — guides discovery, never lectures |
